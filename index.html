Assignment 3 - Git Command Line and First Website

Purpose:

To learn to use basic Linux commands, Vi Text Editor, and Git from the command line. To create a webpage and host it using GitHub Pages.

Linux Commands:

  • pwd – It tells where we are located in the file system.
  • ls – List files. It lists all the files and directories in the current directory.
  • cd “DirectoryName” – Is used to change the directory to “DirectoryName”.
  • dir – It lists all the directories in the current working directory.
  • clear – It clears up the screen.
  • mkdir “DirectoryName”– It creates a new directory by the name “DirectoryName” in the current working directory.
  • git push origin master – Uploads the offline content onto the repository.

Vi Commands:

  • vi “filename” – Creates a file with “filename” in the working directory.
  • i – Turns on the insert mode and edit the file.
  • esc – Exits the insert mode.
  • Shift + : - Opens up the command prompt for writing and other actions.
  • w – writes the text in the file.
  • q – quits the file.
  • wq – writes and quits the file.

Git Commands:

  • git init – Initializes git and turns the current working folder to a git repository.
  • git status – Gets the status of the repository.
  • git add “filename” – Changes to be committed.
  • git commit -m “Comment” – It commits the file to the directory with the “Comment” as -m is used for message.
  • git checkout -b “BranchName” – It switches to the “BranchName” and the “-b” is used to create a new branch with the “BranchName” and creates a copy of the main branch with all the files.
  • git merge “BranchName” – It merges the “BranchName” branch to the main branch.
  • git config --global user.name “Username” - sets a global variable username.
  • git config --global user.email “Email” - sets a global variable email.
